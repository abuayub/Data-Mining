SVD Code:
1. It is a python based code.
2. Installation of Python 3.4.3 is required(I have used Anaconda 2.3 |Spyder 2 Editor)
3. In python shell use command: python svd.py
4. Keep csv files in same folder.

Running:
1. Enter data filename (It is hard coded as iris.csv, can be changed to att.csv).
2. Output csv filename is also hard coded.
3. Run the code to get the 2 dimentional data.
